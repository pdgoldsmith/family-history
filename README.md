# The Goldsmith Family — A Documented History

A well-sourced family history website for the Goldsmith family, tracing the ancestry
of Paul David Goldsmith (born 1983) from Eastern Europe to America.

## Live site

Published at GitHub Pages — [link to be added after first deployment]

## Structure

```
index.html              — Home / family overview
people/
  david-goldsmith.html  — David Goldsmith (great-great-grandfather)
  bluma-levinson.html   — Bluma Levinson (David's sister)
sources.html            — Primary sources index with images
css/style.css           — Stylesheet
js/main.js              — Lightbox and navigation
images/records/         — Scanned primary source documents
```

## Adding records

Copy scanned images to `images/records/` using the naming convention shown in
`sources.html`. Images referenced there will automatically appear in the sources index.
